✅ Task Manager API
A lightweight backend project built with Express.js and Node.js to practice RESTful APIs and CRUD operations. It helps beginners understand routing, request handling, and API testing with Thunder Client or Postman.

🚀 Features
📝 Add and manage tasks

📋 View all tasks with optional status filters

✏️ Update task details

✅ Mark tasks completed

🗑️ Delete tasks individually or clear all completed

🛠️ Tech Stack
Node.js

Express.js

CORS

Thunder Client / Postman (for testing)

📂 Project Structure
Code
task-manager-api/
│── server.js
│── routes/
│   └── tasks.js
│── package.json
⚙️ How to Run
Download or clone the repository

Run npm install to install dependencies

Start the server with npm start

Test endpoints at http://localhost:5000/api/tasks
