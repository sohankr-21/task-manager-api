# ✅ Task Manager API  
A lightweight backend project built with **Express.js** and **Node.js** to practice RESTful APIs and CRUD operations. It helps beginners understand routing, request handling, and API testing with Thunder Client or Postman.

---

## 🚀 Features  
- 📝 Add and manage tasks  
- 📋 View tasks with optional status filters  
- ✏️ Update task details  
- ✅ Mark tasks as completed  
- 🗑️ Delete tasks individually or clear all completed  

---

## 🛠️ Tech Stack  
- Node.js  
- Express.js  
- CORS  
- Thunder Client / Postman

## ⚙️ How to Run  
1. Download or clone the repository  
2. Run `npm install` to install dependencies  
3. Start the server with `npm start`  
4. Test endpoints at **http://localhost:5000/api/tasks**
